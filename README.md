# Primer parcial MCGA

Simple REST API application using MongoDB as a database.


## Installation

Open a terminal, go to the project's root directory and run:

> `> npm install`
More info:
[npm install command](https://docs.npmjs.com/cli/v6/commands/npm-install)

## Run the application

Open a terminal, go to the project's root directory and run:

> `> npm run start`
